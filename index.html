Calcular pagos en cuotas sobre un monto determinado.

Seguir el siguiente procedimiento para iniciar el proceso de conteo de cuotas:

  • -Refrescar la pagina
  • -Ingresar el monto a cacular las cuotas
  • -Ingresar numero de cuotas
  • -Luego el sistema le indicara cuanto debra cancelar mensualmente
  • -Para reiniciar el proceso, refrescar pagina
